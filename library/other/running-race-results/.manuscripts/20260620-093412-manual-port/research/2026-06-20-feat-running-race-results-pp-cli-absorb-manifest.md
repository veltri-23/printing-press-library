# Running Race Results Absorb Manifest

## Built Commands

- `lookup` resolves a fuzzy race name and returns one result by bib or name.
- `athlete` returns cross-event race history from Athlinks or NYRR.

## Verified Novel Features

- Fuzzy race result lookup across multiple timing providers.
- Cross-event athlete history with optional Athlinks current-user token support.
