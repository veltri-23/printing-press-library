# Shipcheck

Run date: 2026-06-21 UTC

## Scope

Paul Graham Essays is a read-only static-site CLI for https://www.paulgraham.com. The source site exposes static HTML, not an official JSON API, so the package ships one archived `GET /articles.html` wrapper plus native commands that parse the index and linked essay pages.

Phase 5 credential validation is intentionally skipped because paulgraham.com is public static content and has no auth flow. All exposed native commands are read-only.
