# Lobsters public JSON print research

Source: public read-only Lobsters JSON endpoints verified against live responses.

Promoted workflows:

- `page`: read the current front page.
- `newest`: read newest submissions by page.
- `hottest-json`: read the hottest feed.
- `t`: read stories for a tag.
- `s`: fetch one story with its comments using a Lobsters short ID.

No authentication is required.
