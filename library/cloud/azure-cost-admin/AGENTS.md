# Azure Cost Admin Agent Notes

`azure-cost-admin-pp-cli` is a read-only Azure FinOps CLI. Keep commands non-interactive and avoid printing Azure subscription IDs, tenant IDs, account emails, raw resource IDs, customer names, or exact spend amounts in public notes.

Use `--dry-run` first when demonstrating request shape. Use `--agent` and `--select` for small machine-readable output.
